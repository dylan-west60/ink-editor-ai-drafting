## What Ink Editor is for

Ink Editor is designed for anyone who writes a lot and wants a streamlined place to create drafts, edit documents, and move between plain text and richer file formats without friction. It combines a distraction-light editor with workflow tools that help keep your attention on the page instead of the UI.

The editor also connects to AI-assisted writing via external APIs, including Claude API and OpenAI API, while staying centered on document creation. With markdown and DOCX support, autosave, multilingual use, and environment profiles, it is meant to adapt to different writing routines without adding unnecessary complexity.

## Using the editor

Begin by opening an existing document or starting a new draft, then select the writing mode that matches your current task. Ink Editor keeps movement and editing close to the keyboard, so everyday actions can usually be handled without leaving the work area.

Typical workflows include:
- Writing in a focused editor layout
- Turning on focus mode when you want fewer distractions
- Importing or exporting content in markdown or DOCX formats
- Using draft map navigation to move between sections
- Enabling AI features through your configured API provider
- Saving different setups with environment profiles for separate projects or contexts

Common configuration areas include:
